3 // Generic classes that model N:N, N:1 and 1:1
4 // relationships. These classes have a myriad of
5 // applications.
6 //
7 // We define a class that maps elements of one Set S1
8 // to another set S2. You can create N:N, N:1 and 1:1
9 // relationships between objects. The only restriction
10 // is that the elements in the relation should be elements
11 // of the Domain Set or Range Set.
12 //
13 // We note that it is possible to access all r in R for a
14 // given d in D as well as all d in D for a given r in R.
15 //
16 // The class 'Set' is an encapsulation of STL 'set' and is easier
17 // to understand
18 //
19 // (C) Datasim Education BV 2004
