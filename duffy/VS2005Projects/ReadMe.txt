These project files are for Visual Studio 2005.
The project include paths asume that you installed the book's source code in the directory "C:\CPP2006\". If you installed the book's source code in a different directory then you must change tha projects additional include paths to your location.

The projects that use MicroSoft Excel asume that Micorsoft Excel 2003 is installed. If another version is installed, change the includes in the "Utilities\ExcelDriver\ExcelImports.cpp" file.
